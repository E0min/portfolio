# WeatherAPI

실시간 날씨 정보 시각화 포스터

## 주요 기능

- **날씨별 동적 배경**: 맑음(태양), 비(빗방울), 눈(발자국), 흐림(구름) 애니메이션
- **실시간 날씨 정보**: Weather API 연동으로 현재 날씨, 온도, 위치 정보 표시
- **날짜 및 시간**: 현재 월/일과 시각 정보 자동 업데이트
- **위치 기반 서비스**: Geolocation API로 사용자 위치의 날씨 데이터 가져오기
