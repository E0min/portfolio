# Firebase Hosting

Firebase 호스팅 웹 프로젝트

## 주요 기능

- **Firebase 호스팅**: 정적 웹사이트를 빠르고 안정적으로 제공
- **글로벌 CDN**: 전 세계 어디서나 빠른 접근 가능
- **자동 HTTPS**: SSL 인증서 자동 적용
- **확장 가능한 구조**: Firebase 생태계와 쉽게 통합
